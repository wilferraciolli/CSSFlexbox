Flexwesome

Coupling of BCs and feature teams - team A may make a change to their schema unaware that team B have a dependency on the same schema. Environment Creation issues - because schema A has views on schema B, schema B must be created first. This means it's not possible to simply ask all BCs to create their schema at the same time. Creation must occur in the right order or errors will occur.
